Learning Objective

Students will build a Vite-based React app, clear boilerplate, render JSX with variables, and apply inline styling.

Exercise: Greeting App

Objective: Create a React app with a dynamic, styled heading and a reusable component.

Steps:

  1. Create a Vite App:
    • Run npm create vite@latest, name it greeting-app, select React, JavaScript.
    • Navigate: cd greeting-app.
    • Install: npm install.
    • Start: npm run dev.
  2. Clear Boilerplate:
    • Delete src/App.css.
    • Empty src/index.css.
    • Simplify App.jsx: 
      function App() {
  return <div></div>;
}
export default App;
  3. Update App.jsx:
    • Add a variable: const greeting = "Welcome to V School!";.
    • Add styles: const styles = { color: "#6b48ff", fontSize: "2em", textAlign: "center" };.
    • Render: 
      function App() {
  const greeting = "Welcome to V School!";
  const styles = { color: "#6b48ff", fontSize: "2em", textAlign: "center" };
  return (
    <div>
      <h1 style={styles}>{greeting}</h1>
    </div>
  );
}
export default App;
  4. Create a Component:
    • Create src/components/Tagline.jsx: 
      function Tagline() {
  return <p style={{ color: "#5039cc", fontSize: "1.2em" }}>Learn. Build. Code.</p>;
}
export default Tagline;
    • Update App.jsx: 
      import Tagline from './components/Tagline';

function App() {
  const greeting = "Welcome to V School!";
  const styles = { color: "#6b48ff", fontSize: "2em", textAlign: "center" };
  return (
    <div>
      <h1 style={styles}>{greeting}</h1>
      <Tagline />
    </div>
  );
}
export default App;
  5. Test:
    • Save and check http://localhost:5173.
    • Verify the greeting and tagline display with purple styling.

Tips: Use DevTools (F12) to debug errors. Ensure import paths are correct. Experiment with styles (e.g., fontSize: "3em").

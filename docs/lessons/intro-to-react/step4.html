Exercise: React Practice

Let’s recap React’s core concepts and assign a practice exercise to solidify your skills.

Key Takeaways

  • Declarative: React simplifies UI updates with a “what, not how” approach.
  • Composable: Reusable components make apps modular and maintainable.
  • JSX: HTML-like syntax with {} for dynamic content.
  • Vite: Fast setup and HMR enhance development.

React Practice

Exercise: React Practice

Objective: Create a simple React app using Vite to practice setup, JSX, and styling.

Instructions:

  1. Create a Vite App:
    • Run npm create vite@latest, name it react-practice, select React, JavaScript.
  2. Clear Boilerplate:
    • Delete src/App.css.
    • Empty src/index.css.
    • Simplify App.jsx: 
      function App() {
  return <div></div>;
}
export default App;
  3. Render an <h1>:
    • Add an <h1> element in App.jsx.
  4. Use Variables:
    • Create a string variable (e.g., const greeting = "Hello, V School!";).
    • Create a style object (e.g., const styles = { color: "#6b48ff", fontSize: "2em" };).
    • Render: <h1 style={styles}>{greeting}</h1>.
  5. Run App:
    • Run npm run dev, view at http://localhost:5173.

Example:

function App() {
  const greeting = "Hello, V School!";
  const styles = {
    color: "#6b48ff",
    fontSize: "2em",
    textAlign: "center",
    margin: "20px"
  };
  return (
    <div>
      <h1 style={styles}>{greeting}</h1>
    </div>
  );
}
export default App;

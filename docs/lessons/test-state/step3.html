Conditionally Rendering and Flipping State (Part 1)

Learning Objectives

By the end of this lesson, students will be able to:

  • Conditionally render the description and rating paragraph elements by toggling state.
  • Conditionally render the details button by toggling state.

Introduction

In this lesson, we'll extend our MovieBuddy app to demonstrate the concepts of conditionally rendering state and flipping state in React. These techniques are essential for creating dynamic and interactive user interfaces.

Conditional rendering and state flipping enable responsive UIs. Misuse of operators like && can lead to errors, such as rendering undefined: 


/* Incorrect */
{showDetails && movies[movieIndex].description} // Crashes if undefined

Use optional chaining (movies[movieIndex]?.description) or checks. Overusing conditionals can also clutter JSX; extract logic to functions for clarity.

Key Concepts

  1. Conditional Rendering: Displaying different content based on the state.
  2. Flipping State: Toggling a boolean state between true and false.

A simple toggle button illustrates these concepts:


import { useState } from "react";

function ToggleButton() {
    const [isActive, setIsActive] = useState(false);

    return (
        <button onClick={() => setIsActive(prev => !prev)}>
            {isActive ? 'Deactivate' : 'Activate'}
        </button>
    );
}

This button toggles state and text, useful for enabling/disabling features, showing conditional rendering basics.

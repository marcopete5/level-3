Blog Props - Assignment

Objective

Your task is to re-create the first page of the Clean Blog sample website. This exercise will help you practice structuring a React application and passing data through components.

Requirements

  1. Component Structure:

    Organize your React app into the following components:

    <Header />
  <Navbar />
<BlogList />
  <BlogPost />
<Footer />
    • Header should include the Navbar component.
    • BlogList should include the BlogPost component.
  2. Data Handling:

    Use the provided array of blog posts to populate the <BlogPost /> component.

    Pass the title, subTitle, author, and date of each blog post as props to the <BlogPost /> component.

  3. Design and Styling:
    • Ensure that the blog text is overlaid on the image and styled consistently with the design of the sample site.

Passing Criteria

  • Your app should correctly render the blog page layout with the provided data.
  • The components should display the blog posts with the appropriate titles, subtitles, authors, and dates.
  • The page should be styled to ensure the text is clearly readable over the images and is visually appealing.

Summary

By completing this assignment, you will gain experience in component-based architecture, data handling in React, and implementing responsive design. Use the provided data to pass props effectively and style the components to match the Clean Blog template’s aesthetic.

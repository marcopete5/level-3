Learning Objectives

Students will create a React app with layout components and submit a practice exercise.

Exercise: Portfolio Layout

Objective: Create a React app with header, navbar, footer, and a Bio component using fragments.

Steps:

  1. Create a Vite App:
    • Run npm create vite@latest, name it portfolio-app, select React, JavaScript.
    • Navigate: cd portfolio-app.
    • Install: npm install.
    • Start: npm run dev.
  2. Clear Boilerplate:
    • Delete src/App.css.
    • Empty src/index.css.
    • Simplify App.jsx: 
      function App() {
  return <div></div>;
}
export default App;
  3. Create Components:
    • src/components/Header.jsx: 
      function Header() {
  return (
    <header style={{ background: "#4a6fa5", color: "#ffffff", padding: "10px", textAlign: "center" }}>
      <h1>My Portfolio</h1>
    </header>
  );
}
export default Header;
    • src/components/Navbar.jsx: 
      function Navbar() {
  return (
    <nav style={{ background: "#e8ecef", padding: "10px" }}>
      <ul style={{ listStyle: "none", display: "flex", justifyContent: "center", gap: "20px" }}>
        <li><a href="#">Home</a></li>
        <li><a href="#">Projects</a></li>
      </ul>
    </nav>
  );
}
export default Navbar;
    • src/components/Footer.jsx: 
      function Footer() {
  return (
    <footer style={{ background: "#2f2f2f", color: "#ffffff", padding: "10px", textAlign: "center" }}>
      <p>Â© 2025 My Portfolio</p>
    </footer>
  );
}
export default Footer;
    • src/components/Bio.jsx: 
      function Bio() {
  const name = "Your Name";
  return (
    <>
      <h2>{name}</h2>
      <p>Web developer learning React at V School.</p>
    </>
  );
}
export default Bio;
  4. Update App.jsx: 
    import Header from './components/Header';
import Navbar from './components/Navbar';
import Footer from './components/Footer';
import Bio from './components/Bio';

function App() {
  return (
    <>
      <Header />
      <Navbar />
      <main>
        <Bio />
      </main>
      <Footer />
    </>
  );
}
export default App;
  5. Test:
    • Run npm run dev, check http://localhost:5173.
    • Verify all components and fragment usage.

Tips: Debug import errors in DevTools (F12). Experiment with styles (e.g., fontSize: "1.5em"). Ensure fragments are used in Bio.

Key Takeaways

  • Custom Components: Functions returning JSX for reusable UI.
  • JSX in App: Import and render components with JSX.
  • Fragments: Group elements without extra DOM nodes.
  • Layout Components: Header, navbar, footer for structured UIs.
  • Separate Functions: Improve modularity and collaboration.

Homework: Component Practice

Objective: Build a React app with a custom component and fragment.

Instructions:

  1. Create a Vite App:
    • Run npm create vite@latest, name it component-practice, select React, JavaScript.
  2. Clear Boilerplate:
    • Delete src/App.css, empty src/index.css.
    • Simplify App.jsx: 
      function App() {
  return <div></div>;
}
export default App;
  3. Create a Component:
    • Create src/components/Profile.jsx: 
      function Profile() {

    <>

export default Profile;

    import Profile from './components/Profile';

function App() {
  return (
    <>

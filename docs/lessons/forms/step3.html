Lesson: Implementing handleSubmit in React

We’ll wire a form to add a movie to the favorites list and clear the form afterward. Below are the key pieces you’ll implement.

1) Initial State

const initState = {
  title: '',
  description: '',
  rating: ''
}

const [formData, setFormData] = useState(initState)

2) Handling Input Changes

function handleChange(e) {
  const { name, value } = e.target
  setFormData(prevFormData => ({
    ...prevFormData,
    [name]: value
  }))
}

3) handleSubmit Function 

function handleSubmit(e) {
  e.preventDefault()          // Prevent full page reload
  addMovie(formData)          // Push current form values to favorites
  setFormData(initState)      // Clear the form
}

Attach it to your form:

<form onSubmit={handleSubmit}>
  {/* inputs bound to formData + handleChange */}
  <button className="btn-primary" type="submit">Add to Favorites</button>
</form>

Overview of Input Types in React

React handles form inputs through controlled components. A controlled component is an input element whose value is controlled by React state. Here's a brief overview of the input types we'll discuss, including how to set predetermined values:

  1. Select Input (<select>)

    • The <select> element allows users to choose from a list of options.
    • How it works: The value of the <select> element is managed through React state. The onChange updates that state.
    • Predetermined Value: Initialize the state with the desired option value (e.g., 'dark') to preselect it.

  2. Radio Input (<input type="radio">)

    • Radio buttons let users select one option from a group (same name, different value).
    • How it works: Control each radio’s checked via state; update state on onChange.
    • Predetermined Value: Set initial state to the default radio’s value (e.g., 'email').

  3. Checkbox Input (<input type="checkbox">)

    • Checkboxes support toggles or multiple selections.
    • How it works: Control the checked prop via state; toggle it on onChange.
    • Predetermined Value: Initialize the boolean state (e.g., true) to start checked.

Code Explanation (General Overview)

Select Input: Use state to drive the dropdown’s value; update it via onChange. Set an initial selection by initializing the state.

Radio Input: Radios share a name and differ by value. Control which is selected via a state value; initialize that state to preselect.

Checkbox Input: Bind the checkbox’s checked to a boolean state. Initialize to true to start checked; toggle on change.

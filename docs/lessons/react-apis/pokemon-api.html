Pokemon List with Random Background Color

Assignment

Project Overview

You are tasked with creating a React application that combines two functionalities:

  1. Fetching and displaying Pokémon data.
  2. Managing and updating the background color of the page.

Requirements

1. Fetch and Display Pokémon Data

  • Endpoint: Use https://api.vschool.io/pokemon to get a list of Pokémon.
  • Data Display: Render each Pokémon’s name on a separate line in the browser.
  • Fetching Logic: Use a useEffect hook to fetch the Pokémon data when the component mounts.

2. Manage Background Color

  • Initial Color: Set the background color of the page to a random color.
  • Random Color API: Use https://random-color.onrender.com/colors/random to get a random color.
  • Color Update: Implement a button that, when clicked, fetches a new random color and updates the background color of the page.
  • State Management: Use a state variable to manage the background color and another state variable to control the color update logic.

3. React Hooks

  • useEffect: Implement two useEffect hooks:
    • One for fetching and storing Pokémon data.
    • Another for fetching and updating the background color based on a state variable.
  • useState: Use state variables to manage Pokémon data, background color, and the count for triggering color updates.

4. Component Structure

  • Display Pokémon: Render Pokémon names in a list format.
  • Button Functionality: Include a button to trigger background color changes.

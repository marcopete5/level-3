Overview of Input Types in React

React handles form inputs through controlled components. A controlled component is an input element whose value is controlled by React state. Here's a brief overview of the input types we'll discuss, including how to set predetermined values:

  1. Select Input (<select>)
    • The <select> element allows users to choose from a list of options, commonly used for dropdown menus.
    • How it works:
      • The value of the <select> element is managed through React state. The value attribute is set to a state variable that determines the currently selected option.
      • Predetermined Value: Initialize the state with the desired option value. For example, to set "Dark" as the default selection, initialize the theme state with 'dark'.
  2. Radio Input (<input type='radio'>)
    • Radio buttons allow users to select one option from a group. They are grouped by having the same name attribute but different value attributes.
    • How it works:
      • Each radio button in the group shares the same name attribute and has a unique value. The checked attribute is based on the state value.
      • Predetermined Value: Set the initial state to the value of the radio button you want selected by default. For instance, to select "Email" initially, initialize the contact state with 'email'.
  3. Checkbox Input (<input type='checkbox'>)
    • Checkboxes allow users to select one or multiple options, often for binary choices (true/false) or multiple selections.
    • How it works:
      • The checked attribute is controlled by the state. When toggled, the state updates to reflect the new checked status.
      • Predetermined Value: Initialize the state to true or false to determine whether the checkbox is checked or not by default. For example, setting canContact to true will make the checkbox checked initially.

UseState Practice - Assignment

Assignment

Learning Objectives

  • Practice updating React state with the useState hook.
  • Toggle boolean and string state values and apply conditional UI updates.
  • Update arrays and objects in state without mutation.
  • Verify state updates via expected console outputs.

Objective

The goal of this assignment is to practice updating state in React. This is a fundamental skill for managing and manipulating data within your components. By completing this exercise, you will gain confidence in using the useState hook to handle various state updates.

Instructions

1. Clone the Repository

git clone https://github.com/VSchool/useState-practice.git

2. Remove the .git Directory

cd useState-practice
rm -rf .git

3. Install Dependencies

npm install

4. Implement the Following State Updates

  1. Set a New Color: Pass in a new color of your choosing in place of the old one. We donâ€™t care what the previous color was. Expected Output: The console should log the new color you chose (e.g., "blue").
  2. Toggle Between Two Colors: Toggle between the colors pink and blue. If the current color is pink, set it to blue. If it's currently blue, set it back to pink. Expected Output: Console alternates between "pink" and "blue".
  3. Add a New Person to the Array: Add a new person object to the people array in state (must include firstName and lastName). Final array should contain two person objects. Expected Output: [ { firstName: "John", lastName: "Smith" }, { firstName: "Jane", lastName: "Doe" } ]
  4. Multiply the Number by 2: Update numeric state to be doubled. Expected Output: 8 (if initial state was 4).
  5. Add a New Property to an Object: Add an age property to the existing person state object (do not overwrite other properties). Expected Output: { firstName: "John", lastName: "Smith", age: 30 }
  6. Fix an Incorrect State Update: Correct the following so it adds a new color to the colors array: 
    const [colors, setColors] = useState(["pink", "blue"]);

setColors("green"); // Fix this state update
    Expected Output: ["pink", "blue", "green"]

Verify Your Work

Check the browser console output. The assignment is considered correct when the console.log statements in the code display the expected data for each update described above.

Reason for Practice

Understanding how to update state is crucial for building interactive and dynamic React applications. This exercise covers common state-management scenarios: setting new values, toggling values, adding items to arrays, multiplying numerical state, updating objects without overwriting existing properties, and correctly updating arrays.

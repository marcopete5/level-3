DJ React - Assignment

Assignment

Objective

Create a DJ color board application where you can control the colors of a 2x2 grid of squares using buttons. This exercise will help you practice managing state, handling events, and using CSS Grids in React.

Requirements

  1. Grid Setup:
    • Create a parent component that contains a 2x2 grid of squares.
    • The state of this component will maintain an array of colors: 
    ["white", "white", "white", "white"];
    • Render the squares dynamically by mapping over this colors array and passing the appropriate color to each <Square /> component.

Levels of Advancement

Small Time DJ

  • Add a button to change the color of all four squares.
  • Change all squares to black if the first square is white.
  • Change all squares to white if the first square is black.

Party DJ

  • Add a second button to change the top half of the grid (the two squares on the top) to purple.

Professional DJ

  • Add two more buttons, bringing the total to four.
  • One button should change the color of the bottom left square to blue.
  • The other button should change the color of the bottom right square to blue.

Big Time DJ

  • Add four additional buttons, making a total of eight buttons.
  • Each button should control one of the squares individually.
  • You can decide on the effects each button has on its corresponding square.

Design and Styling

  • Use CSS Grid to layout the 2x2 grid of squares.
  • Style the squares and buttons to be visually appealing and functional.

Passing Criteria

  • The grid should update correctly based on button interactions.
  • Each level of advancement should build upon the previous requirements.
  • The final implementation should include functional buttons and responsive color changes.

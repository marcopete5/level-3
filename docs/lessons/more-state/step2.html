MovieBuddy App: Conditionally Rendering and Flipping State

In this lesson, we'll extend our MovieBuddy app to demonstrate conditionally rendering UI and flipping boolean state in React.

Add a showDetails state

const [showDetails, setShowDetails] = useState(false);

Create a function to toggle showDetails

function toggleDetails() {
  setShowDetails(prevShowDetails => !prevShowDetails);
}

Conditionally render description & rating and toggle button text 

return (
  <div>
    <h1>{movies[movieIndex].title}</h1>

    {showDetails && (
      <>
        <p>Description: {movies[movieIndex].description}</p>
        <p>Rating: {movies[movieIndex].rating}</p>
      </>
    )}

    <button onClick={() => nextMovie(-1)}>Prev Movie</button>
    <button onClick={() => nextMovie(1)}>Next Movie</button>
    <button onClick={getRandomMovie}>Random Movie</button>

    <button onClick={toggleDetails}>
      {showDetails ? "Hide Details" : "Show Details"}
    </button>
  </div>
);

Detailed Concepts

&& Operation

{condition && <JSXElement />}

If condition is true, the element renders; otherwise nothing renders.

Ternary Operator

condition ? <TrueJSXElement /> : <FalseJSXElement />

Use for toggling between two different UI elements (e.g., button label).

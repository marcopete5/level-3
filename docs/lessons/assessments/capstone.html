Meme Generator Project

Level 3 Capstone

Before beginning, consider watching these workshops as a review of the content needed to complete this assignment

  • Vite Setup / Simple Form
  • handleChange / handleSubmit
  • Conditional Rendering
  • React State Management and Prop Drilling

You will make an app that satisfies the following user stories. It will be similar to what was completed in Scrimba, but will be done in VS Code using git:

  • A user will see a meme image on page load
  • A user can click "refresh meme image" to load a new one

2 Hours ~

  • A user will see a form with two inputs and a submit button
  • Inputs will be:
    • top text
    • bottom text
  • A user will be able to see a preview of their meme

2 Hours ~

  • A user will be able to add created memes to a list
  • A user will be able to delete memes from the list

2 Hours ~

  • A user will be able to edit an existing meme

2 Hours ~

Warning: The project will be maintained on GitHub. It is suggested you read and watch everything in Resources before starting the project.

Additional Tips:

  1. Do not set the ID from the meme image as the ID for each created meme. This will cause issues if 2 or more memes use the same image. You can either use the index number from the meme list as the ID or use a UUID generator package.
  2. Perform an HTTP request to https://api.imgflip.com/get_memes to get a list of memes.

Project - Business Time with React

Task Overview

You have been tasked with creating a simple React project using Vite. The goal is to build and render a basic web page for a fake business. This project will help you practice building and styling components in React.

Learning Objectives

Students will create and style React components, applying className to build a structured webpage.

Project: Business Time with React

Objective: Build a React app for a fake business with Header, Main, and Footer components, styled using className.

Requirements:

  1. Create Three Components:
    • Header: Top section of the webpage.
    • Main: Central part of the webpage.
    • Footer: Bottom section of the webpage.
  2. Render All Three Components:
    • Ensure Header, Main, and Footer are rendered correctly.
  3. Style the Components:
    • Position Header at the top, Main in the middle, Footer at the bottom.
  4. Fill Each Component with Content:
    • Header: Business name and navigation menu.
    • Main: Business description, services, and relevant info.
    • Footer: Contact info, business hours, and other details.

Additional Notes:

  • Content: Be creative! The business can be a coffee shop, tech startup, or anything else.
  • Navigation Links: Links in Header are for appearance only, not functional.
  • Tips: Use DevTools to inspect class attributes. Experiment with CSS (e.g., padding: 30px). Ensure className is used correctly.

Key Takeaways

  • JSX: HTML-like syntax in JavaScript, transpiled by Babel.
  • className: Replaces class to avoid JavaScript conflicts.
  • Errors: Fix issues like using class, typos, or missing imports.

What is JSX?

JSX (JavaScript XML) is a syntax extension that lets you write HTML-like code within JavaScript, used in React to define component UI.

Analogy: JSX is like a recipe for your UI, mixing ingredients (HTML-like tags) with directions (JavaScript) to create a component.

Key Features

  • HTML-like Syntax: Familiar and intuitive for web developers.
  • JavaScript Integration: Use {} for dynamic expressions (e.g., variables, functions).
  • Transpilation: Babel converts JSX to JavaScript for browsers.

Example

A simple JSX component:

function App() {
  const message = "Hello, React!";
  return (
    <div>
      <h1>{message}</h1>
      <p>JSX combines UI and logic.</p>
    </div>
  );
}
export default App;

Why JSX?

  • Readability: Unifies UI structure and logic.
  • Dynamic UI: Enables real-time updates via JavaScript.
  • Efficiency: Eliminates separate HTML files.

Pitfall

JSX requires a single parent element (e.g., <div> or <></>). Multiple top-level elements cause errors.

MovieBuddy App - Introduction to State in React

Overview

In this section, we'll be creating a MovieBuddy app. This app allows users to:

  • Get a random movie from the data
  • Create a favorites list
  • Add or delete movies from the favorites list

The primary goal of this section is to introduce one of the most powerful tools in React: State.

Key Concepts

  1. State Management:
    • State is a built-in object that is used to contain data or information about the component.
    • State in React is mutable, meaning it can change over time.
  2. Random Movie Selection: Use state to store and display a randomly selected movie from the data.
  3. Favorites List:
    • Use state to manage a list of favorite movies.
    • Add movies to the favorites list.
    • Delete movies from the favorites list.

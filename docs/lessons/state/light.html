Lightswitch - Assignment

Assignment

Learning Objectives

  • Understand how to manage boolean state in a React component.
  • Learn how to use the useState hook to toggle state.
  • Practice creating a function to update state based on user interaction.
  • Update the UI dynamically based on state changes.

Requirements

  • Set up a new React project using Vite.
  • Create a LightSwitch component with state management using useState.
  • Implement functionality to toggle between light and dark modes.
  • Ensure the UI reflects the current mode with appropriate styling.

Assignment Steps

Create a Basic Vite React App

  1. Set up a new React project using Vite.

Create a LightSwitch Component

  1. Inside your React app, create a new component named LightSwitch.
  2. Add a div containing:
    • An h2 to display the current mode (e.g., “Light” or “Dark”).
    • A button to toggle between light and dark modes.

Implement State Management

  1. Use the useState hook to create a state variable called isDarkMode.
  2. Initialize isDarkMode to false (light mode).

Update the UI on Button Click

  1. Ensure that clicking the toggle button switches isDarkMode between true and false.
  2. Apply different styles (e.g., background and text colors) to the div based on isDarkMode.
  3. The h2 should reflect the current mode, and the UI should update instantly after each toggle.

Expected Outcome

  • A fully functional LightSwitch component where the current mode is displayed in an h2.
  • Clicking the toggle button switches between light and dark modes, updating the UI’s appearance.
  • State management is handled using React’s useState hook, with clear and concise code.

Tips

  • Use a function to toggle isDarkMode that correctly updates the boolean state (e.g., using the previous state value).
  • Apply styles conditionally using class names (e.g., className={isDarkMode ? 'dark' : 'light'}) or inline styles.
  • Ensure state updates are handled asynchronously by using functional updates to avoid stale state issues.

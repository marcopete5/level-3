Practice Assignment: useState Practice in React

Learning Objectives

  • Understand how to update state in a React component using the useState hook.
  • Practice setting state with new values, toggling values, and modifying arrays and objects.
  • Learn to update state immutably to avoid overwriting existing data.
  • Verify state updates through console output.

Reason for Practice

Updating state is crucial for building interactive React applications. This assignment covers common state management scenarios, including setting new values, toggling between values, adding to arrays, modifying numbers, updating objects, and fixing incorrect array updates.

Requirements

  • Create a new React project using Vite.
  • Use the provided starter code in App.jsx and implement the required state updates.
  • Ensure console output matches the expected results for each state update.
  • Add buttons to trigger each state update and display current state values in the UI.

        {
            firstName: 'John',

        }

    const [number, setNumber] = useState(4);

Assignment Steps

Create a Basic Vite React App

  1. Create a new React project using Vite.

Install Dependencies

Implement State Updates

  1. Set a New Color: Add code to set color to a new color of your choice (e.g., "blue"). The console should log the new color.
  2. Toggle Between Two Colors: Add code to toggle toggleColor between "pink" and "blue". The console should alternate between logging "pink" and "blue".
  3. Add a New Person to the Array: Add code to append a new person object (with firstName and lastName) to the people array. The console should log an array with two person objects.

  6. Fix an Incorrect State Update: Fix the setColors('green') call to correctly add "green" to the colors array. The console should log ["pink", "blue", "green"].

Add UI Controls

Verify Your Work

Tips

  • Use functional updates (e.g., prev => [...prev, newItem]) for array and object state to ensure immutability.
  • Place console.log statements inside your update functions to capture the new state value.
  • For the colors array fix, ensure you spread the existing array instead of replacing it.
  • Use CSS to style buttons and state displays for clarity.

JSX and Vite Setup

This section covers JSX, a syntax for writing HTML-like code in JavaScript, and Vite, a fast build tool for React apps.

Learning Objectives

Students will be able to explain JSX, install a Vite-based React app, clear boilerplate, and use curly braces for dynamic JSX.

What is JSX?

JSX (JavaScript XML) lets you write HTML-like code in JavaScript, combining UI structure and logic for intuitive React components.

Benefits

  • Readable: Resembles HTML, familiar to web developers.
  • Dynamic: Use curly braces {} for JavaScript expressions.
  • Efficient: No switching between HTML and JS files.

JSX Examples

Basic JSX:

const element = <h1>Hello, world!</h1>;

function App() {
  return (
    <div>
      {element}
      <p>This is JSX.</p>
    </div>
  );
}

Dynamic content with curly braces:

function Profile() {
  const name = "John Doe";
  const age = 25;
  return (
    <div>
      <h2>{name}</h2>
      <p>Age: {age}</p>
      <p>Time: {new Date().toLocaleTimeString()}</p>
    </div>
  );
}

Advanced Example: Rendering a list:

function ItemList() {
  const items = ["Apple", "Banana", "Orange"];
  return (
    <ul>
      {items.map(item => <li>{item}</li>)}
    </ul>
  );
}

Common Pitfall

Curly braces accept expressions (e.g., {1 + 2}), not statements (e.g., {if (true) {}}). Debug syntax errors in DevTools.

Vite Setup

Vite offers fast server start and Hot Module Replacement (HMR) for React development.

Steps

  • Run: npm create vite@latest.
  • Prompts:
    • Project name: e.g., my-react-app.
    • Framework: Select React.
    • Variant: Choose JavaScript.
  • Navigate: cd my-react-app.
  • Install: npm install.
  • Start: npm run dev.
  • Open: http://localhost:5173.

Clear Boilerplate

Simplify the Vite app:

  • Delete src/App.css.
  • Empty src/index.css.
  • Update src/App.jsx: 
    function App() {
  return <div></div>;
}
export default App;

Tips

Ensure Node.js v16+ (node -v). If port 5173 is busy, try 5174. Verify the Vite logo in the browser.

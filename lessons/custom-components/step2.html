Creating and Using Custom Components

Learn how to create custom components, add JSX to App, and organize them for reusability.

Learning Objective

Students will define and use custom components with JSX in App.

Steps to Create a Component

  1. Create a File: In src/components, create a file (e.g., MyComponent.jsx) using PascalCase.
  2. Define the Component: Write a function returning JSX.
  3. Export: Use export default for importing elsewhere.

Example: Simple Component

// src/components/MyComponent.jsx
function MyComponent() {
  return (
    <div>
      <h2>Hello, World!</h2>
    </div>
  );
}
export default MyComponent;

Adding JSX to App

Import and use the component in App.jsx, as provided: 

// src/App.jsx
function App() {
  //JS
  return (
    //JSX
    <>
      <h1>Why Learn React?</h1>
      <ol>
        <li>Its a popular library, so I'll be in the know</li>
        <li>I'm more likely to get a job if I know React</li>
      </ol>
    </>
  )
}
export default App;

Main File:

// src/main.jsx
import React from 'react';
import ReactDOM from 'react-dom/client';
import App from './App.jsx';
import './index.css';

ReactDOM.createRoot(document.getElementById('root')).render(
  <React.StrictMode>
    <App />
  </React.StrictMode>,
);

Dynamic Example

A Welcome component with JSX:

// src/components/Welcome.jsx
function Welcome() {
  const message = "Learn React!";
  return (
    <div>
      <h2>Welcome to Our Site!</h2>
      <p>{message}</p>
    </div>
  );
}
export default Welcome;

// src/App.jsx
import Welcome from './components/Welcome';

function App() {
  return (
    <>
      <h1>React App</h1>
      <Welcome />
    </>
  );
}
export default App;

Pitfalls

  • Naming: Use PascalCase (e.g., MyComponent) to avoid confusion with HTML tags.
  • Exports: Missing export default causes “Component not found” errors.
  • JSX: Ensure one parent element (use fragments if needed).
